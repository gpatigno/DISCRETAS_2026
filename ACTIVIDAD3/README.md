
# **Actividad de Clase**
## Verificación de Documentos con Firma Digital RSA

### **Objetivo**
    
### Comprender y aplicar el proceso de firma digital con RSA para verificar la autenticidad e integridad de documentos PDF.


### Introducción rápida
- En los sistemas modernos, la firma digital permite:
    * Verificar la autenticidad del emisor
    * Garantizar la integridad del documento


### Preguntas orientadoras
- Qué es el hash de cualquier archivo o documento, y para qué sirve?
- Cómo se firma un documento mediante RSA?
### Proceso para la firma digital
- Se calcula el hash del documento.
- Se firma el hash con la clave privada.
- Se verifica con la clave pública.
